# دليل استعادة نظام واتساب (WhatsApp Recovery Guide) 🛠️

هذا الدليل مخصص للمطور أو الأدمن للتعامل مع حالات انقطاع خدمة البوت.

## 1. كيفية إعادة الربط (Re-linking)
إذا فقد البوت الاتصال أو ظهرت رسالة `Logged Out`:
1. أوقف السيرفر.
2. احذف مجلد `server/auth_info_baileys`.
3. أعد تشغيل السيرفر.
4. سيظهر QR Code جديد في الـ Terminal، قم بمسحه باستخدام واتساب الهاتف.

## 2. التأكد من حالة الاتصال
يمكنك التحقق من حالة النظام بزيارة الرابط الرئيسي:
`GET /` (على رابط السيرفر المرفوع)

**الحالات المتوقعة:**
- `HTTP 200 OK`: النظام يعمل بالكامل والواتساب متصل.
- `HTTP 503 Degraded`: السيرفر يعمل ولكن الواتساب غير متصل أو معطل.

**ملاحظة لبيئة Render:**
إذا كنت تستخدم **Persistent Disk**، فمسح الجلسة يتطلب الدخول لـ Render Dashboard ومسح محتويات الـ Disk أو إعادة تهيئته.

## 3. تعطيل الخدمة مؤقتاً (Emergency Stop)
إذا حدثت مشكلة في حساب واتساب وتريد إيقاف الإرسال مؤقتاً دون إيقاف السيرفر:
1. اذهب لملف `.env`.
2. غيّر القيمة إلى `ENABLE_WHATSAPP=false`.
3. أعد تشغيل السيرفر. سيستمر الموقع في العمل ولكن بدون إرسال رسائل واتساب.

## 4. فحص الأخطاء (Logs)
جميع الأخطاء مسجلة بتفصيل تقني في:
- `server/logs/error.log` (للأخطاء الفادحة)
- `server/logs/combined.log` (لجميع الأحداث)

استخدم الـ `requestId` الموجود في الاستجابة للبحث عن الخطأ داخل الملفات.
